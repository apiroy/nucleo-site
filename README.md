# nucleoai.site Landing Page

Sitio web para agencia de desarrollo de plataformas e-commerce a medida, enfocada en emprendedores +40.

## Características
- Diseño premium tipo n8n.io.
- Sin dependencias de Tailwind (Vanilla CSS Modules).
- Optimizado para SEO.
- Tres niveles de precios claros.

## Tecnologías
- Next.js 15 (App Router).
- TypeScript.
- Vanilla CSS (Modulos).
- Google Fonts (Inter, Manrope).
- Material Symbols Outlined.

## Estructura
- `/app`: Páginas y layouts.
- `/app/globals.css`: Sistema de diseño (tokens).
- `/app/page.module.css`: Estilos específicos de la landing.
